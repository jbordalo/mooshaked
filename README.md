# Mooshaked

## About The Project

Check for invalid characters in your program before you submit it to Mooshak.

## Getting Started

`git clone https://github.com/jbordalo/mooshaked.git`

`cd mooshaked`

`npm install`

## Running

`npm run dev`

## Building

`npm run build`
